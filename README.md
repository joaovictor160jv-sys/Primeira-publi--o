🛍️ Easy Shop
📌 Sobre o projeto

O Easy Shop é uma landing page criada com o objetivo de praticar desenvolvimento front-end, focando em estruturação com HTML e estilização com CSS.

O projeto apresenta um layout moderno e adaptável para diferentes tamanhos de tela (responsivo), simulando uma página de apresentação de um serviço de compras online.
